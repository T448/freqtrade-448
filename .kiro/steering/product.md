# Product Overview

## Product Overview

Freqtrade is a free and open-source cryptocurrency trading bot designed to automate trading strategies across multiple cryptocurrency exchanges. It provides a comprehensive framework for algorithmic trading with built-in risk management, backtesting capabilities, and machine learning integration for strategy optimization.

## Core Features

- **Multi-Exchange Support**: Compatible with 15+ major cryptocurrency exchanges (Binance, Bybit, Gate.io, HTX, Hyperliquid, Kraken, OKX, etc.)
- **Strategy Framework**: Flexible Python-based strategy development with technical indicators
- **Backtesting Engine**: Comprehensive historical data testing with detailed performance metrics
- **Risk Management**: Built-in protections, stop-loss mechanisms, and position sizing controls
- **Machine Learning Integration**: FreqAI module for strategy optimization using ML models
- **Real-time Monitoring**: Web UI, Telegram bot, and REST API for remote control and monitoring
- **Futures Trading**: Experimental support for leveraged futures trading across supported exchanges
- **Data Management**: Historical data collection, storage, and analysis tools
- **Hyperparameter Optimization**: Automated strategy parameter tuning using various optimization algorithms

## Target Use Case

### Primary Users

- **Experienced Python Developers**: Users with coding knowledge who want to implement custom trading strategies
- **Quantitative Traders**: Professionals seeking to automate and optimize their trading algorithms
- **Educational Users**: Individuals learning algorithmic trading concepts (with strong risk disclaimers)
- **Strategy Researchers**: Analysts who need comprehensive backtesting and optimization tools

### Specific Scenarios

- **Automated Portfolio Management**: Running 24/7 trading strategies across multiple pairs and exchanges
- **Strategy Development**: Building, testing, and refining trading algorithms with historical data
- **Risk Management**: Implementing sophisticated position sizing and protection mechanisms
- **Multi-Market Arbitrage**: Executing strategies across different exchanges and trading pairs
- **Research and Analysis**: Studying market behavior and strategy performance using built-in analysis tools
